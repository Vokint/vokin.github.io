Java 是一種十分基礎並常用的編程語言。 它的歷史非常悠久，在許多新語言的研究和開發中擔任著重要的角色, 但亦因此 Java 比其他常用語言更為繁瑣和嚴謹。 不過由於 Java 穩定的表現, 豐富的文檔和輕鬆搬移項目的靈活特性，至今仍然有不少從事資訊科技的公司採用。

注意事項

Java 十分重視句式結尾,千萬不要忘記在最後加上分號 ; (儘管不少IDE會自動提醒你)


如何在不使用開發軟件的前提下,單獨運行 Java 檔案

  1. 首先你的裝置上必須安裝了 JDK
  2. 手上持有 Java 檔案,例: [檔案名稱].java
  3. 打開命令提示元 CMD 並輸入 cd [檔案位置] 將目標位置設定在檔案文件夾
  4. 輸入 javac [檔案名稱].java 以創建執行 Java 用的檔案 [檔案名稱].class
  5. 最後在命令提示元中輸入 java [檔案名稱].java 即可

如何單獨運行 Java 檔案並附帶參數 Parameter

  1. 首先依照上述的指示做一遍
  2. 在最後一步的指令後方添加任意Parameters
    java [檔案名稱].java <parameter1> <parameter2> <…>
    所有Parameters將會儲進一個Array並傳遞至main裡面的String[] args

章節二 - Objects Oriented Programming (OOP)

Java 開發的核心基本圍繞在 Object, 而 Object 的原型就是 Class

甚麼是 Class

簡單來說 Class 是一個包含虛化定義的 Template, 在實體化後就變成 Object
舉個例子

Class type 種類

不同的種類會導致不同的存取權限
分類不限於 Class ,更能用在 Variable 和 Method 上

  • Public (最大權限)
    Project內任何檔案都能存取
  • Protected
    同一 Class, Package, 其他 Package 的 Subclass 能存取
  • Default
    僅限同一 Class, Package 能存取
  • Private (最小權限)
    僅限同一 Class 能存取
